# SimpleViewer

画像ビューアアプリケーション (.NET 8 / WPF)

## 機能

- 画像ファイルの表示
- Exif情報の表示 (SimpleExifLib を使用、※未実装)
- 画像の向きの自動調整 (Exifデータに基づく)

## 使用技術

- .NET 8
- WPF (Windows Presentation Foundation)
- SimpleExifLib (カスタムライブラリ)

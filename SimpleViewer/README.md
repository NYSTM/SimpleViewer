# SimpleViewer

SimpleViewerはシンプルなWPFベースのイメージビューアです。
フォルダ、ZIP、PDF、単一の画像ファイルを開いてページ表示、見開き表示、サムネイルやツリーでのナビゲーションが可能です。

## 特徴

- フォルダ、ZIP、PDF、単一画像の読み込み
- ページ表示・見開き表示
- サムネイル表示とツリーナビゲーション（仮想化により多数ページでも高速）
- ドラッグ＆ドロップでのファイル読み込み
- ズーム、パン（ドラッグ操作）、クリックによるページ遷移
- カタログオーバーレイ表示

## 主な操作

- 左端をクリック：次ページへ移動（View の左 20% を端領域としています）
- 右端をクリック：前ページへ移動（View の右 20% を端領域としています）
- 画像領域をドラッグ：パン（スクロール）
- スライダーやサムネイル、ツリーパネルで直接ページ指定が可能

## キーボードショートカット（代表例）

- `Ctrl+O` : 開く
- `Ctrl+W` : 閉じる
- `F3` : カタログ表示
- `F4` : サイドバー切替
- `S`  : 表示モード切替
- `Ctrl++` / `Ctrl+-` : ズームイン / ズームアウト
- `Ctrl+0` : 実寸（100%）
- `Ctrl+F` : ウィンドウ幅に合わせる
- `Ctrl+G` : ウィンドウ全体に合わせる

> 注：動作の詳細（端領域の幅やクリック判定閾値）はコード内で設定されています。必要に応じて調整可能です。

## 必要条件

- Windows（WPFアプリケーションのため）
- .NET 8 SDK

## ビルドと実行

1. リポジトリをクローンします。
2. コマンドラインでプロジェクトルートに移動します（`SimpleViewer.csproj` があるディレクトリ）。
3. 依存関係を復元してビルドします:

   `dotnet restore`

   `dotnet build -c Release`

4. 実行します:

   `dotnet run --project SimpleViewer.csproj`

あるいは、Visual Studio 2022/2023 でプロジェクトを開き、通常の手順でデバッグ/実行してください。

## 使い方

1. SimpleViewerを起動します。
2. 「ファイル」メニューから「開く」を選択し、表示したいフォルダ、ZIPファイル、PDFファイル、または画像ファイルを選択します。ドラッグ＆ドロップでも開くことができます。
3. 画像が表示されます。ツールバーやメニューから表示モードを切り替えたり、ナビゲーションパネルを使用してページを移動できます。
4. 必要に応じて、ズームイン・ズームアウトやフルスクリーン表示も利用できます。
5. 終了するには、「ファイル」メニューから「終了」を選択します。
